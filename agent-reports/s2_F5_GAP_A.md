**CLAIM 1 (Lemma α′ — tilted-sea staircase interface bound). PROVED. Confidence 0.93.**
Let θ ∈ (0, π/4], τ = tan θ, and φ₀ ≥ 0 with **tan φ₀ ≤ τ/2**. Let η ≤ 1/20, h = 1/4. Let Q be a finite family of squares with sides in [1−η, 1+η] and **folded tilts ≤ φ₀** (not necessarily axis-parallel), pairwise disjoint interiors, all contained in {y ≤ τx + β}. Then for every X ≥ 1 the strip R = {0 ≤ x ≤ X, τx+β−h < y < τx+β} satisfies
**|R \ ∪Q| ≥ (X/228)·min(τ − tan φ₀, 1/3) ≥ (X/456)·min(τ, 2/3).**

*Proof (complete; §1 of derivations).* The assignment predicted the original crest/quantization argument would generalize; it does, and in fact **simplifies** — crest uniqueness (the quantization step of Lemma α) is not needed at all:
(1) *Vertical accounting.* F(x) := max{T_S(x) : S spans x} where T_S(x) is S's upper envelope; any covering square has top ≥ the covered point, and all tops ≤ ℓ(x) := τx+β; so uncovered ≥ ∫₀^X min(h, ℓ−F).
(2) *Per-square envelope (the new step).* For a square with signed folded tilt ψ ∈ [−φ₀, φ₀], the upper envelope is concave with slopes tan ψ then −cot ψ (ψ ≥ 0), or cot|ψ| then −tan|ψ| (ψ < 0). Anchoring at the leftmost vertex (ψ ≥ 0) or topmost vertex (ψ < 0) and using that the anchor lies under the line: **ℓ(x) − T_S(x) ≥ τ₁·|x − a_S| on the whole span**, τ₁ := τ − tan φ₀. The ψ<0 left branch needs cot|ψ| − τ ≥ τ₁, which follows from tan φ₀ ≤ τ/2 ≤ 1/2 and τ ≤ 1 (cot|ψ| ≥ 2/τ, 2/τ − τ ≥ 1 ≥ τ₁). This is where tan φ₀ ≤ τ/2 enters, and it is the sharp shape: at φ₀ = θ a coherent row hugs the line at zero cost.
(3) *Crest = argmax decomposition + V-shaped bathtub:* ∫_{I_j} min(h, τ₁|x−a_j|) ≥ (τ₂/4)|I_j|², τ₂ := min(τ₁, 2h/w_max) ≥ min(τ₁, 1/3), w_max = (1+η)(cos φ₀+sin φ₀) ≤ 1.409.
(4) *Crest count:* crest squares lie in a sheared parallelogram of area 3.23(X+2.98); disjointness gives J ≤ 3.58X + 10.67 ≤ 14.25X (X ≥ 1).
(5) *Assembly:* Cauchy–Schwarz + the |N₀| ≥ X/2 dichotomy gives the box. ∎
Numerics: adversarial families (coherent +φ₀ rows hugging the line — the optimal cheat, coherent −φ₀, axis staircases, random mixed tilts; all SAT-verified disjoint and below the line), τ ∈ {0.1, 0.2, 0.4, 1}, tan φ₀/τ ∈ {0, ¼, ½}, X = 20: **minimum ratio measured/bound = 112.3, zero violations**; the tightest configuration is exactly the predicted coherent hug.
